💡 Cara Menambah Cookie
Langkah 1: Install ekstensi Cookie-Editor di browser (Chrome/Firefox).

Langkah 2: Login ke akun Netflix kamu di browser.

Langkah 3: Buka Cookie-Editor → klik ikon Export → pilih "Export as Netscape".

Langkah 4: Copy semua teks yang muncul → paste di halaman Kelola Cookie → Tambah Cookie.

Langkah 5: Isi metadata (opsional) → klik Simpan. ✅

💡 Kamu juga bisa tambahkan baris metadata di atas cookie:
name: Nama Profil Netflix
plan: Premium
country: ID
next billing: 2026-07-25
